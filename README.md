# Wild_Root_Prompt — Technique Browser

A read-only browser for the 173 prompt engineering techniques (15 categories —
Chain-of-Thought, Tree-of-Thought, ReAct, MECE, red teaming, and more) used by
[Wild_Root_Prompt](https://github.com/TFD-42/Wild_Root_Prompt).

This Space has **no Ollama dependency** — it can't reach your local machine,
so it's a catalog, not a generator. The actual tool runs 100% locally via
Ollama: no cloud API keys, nothing leaves your machine.

- [View the full project on GitHub](https://github.com/TFD-42/Wild_Root_Prompt)
- [Install it locally](https://github.com/TFD-42/Wild_Root_Prompt#quick-start)
